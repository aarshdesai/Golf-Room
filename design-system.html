Dev Handoff · Design System

The Golf Room

All design tokens and components for the Golf Room booking platform. Every value references a CSS variable from globals.css — no hardcoded values anywhere.

Colors

All colors defined as CSS custom properties. Light mode is default; dark mode applied via [data-theme="dark"] on <html>.
