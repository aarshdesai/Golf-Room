Page Wireframes

01
Landing landing.html
Marketing homepage — hero, features, how it works, bay showcase, membership tiers, testimonials, and footer. Sign-in/register is a modal (no separate auth page).
Hero split-layout Auth modal (tabbed) Membership tiers Testimonials
02
Bay Browser bays.html
Immersive bay selection grid with multi-field filter bar (date, time, players, type), active filter chips, and a 440px detail sidebar.
Filter bar + chips 2-col bay grid Detail sidebar 6 named palettes
03
Booking Flow booking.html
Full 3-step booking wizard in a single page (step divs toggled). Step 1 = calendar grid with color-coded bay chips; Step 2 = bay selection; Step 3 = review. Sticky bottom bar shows live selection summary.
3-step wizard Calendar grid Bay filter chips Sticky selection bar
03a
Date & Time Picker booking-datetime.html
Standalone Step 1 detail view — calendar panel with selected slot callout sidebar, price preview, and free cancellation note.
Standalone Step 1 Slot callout Price preview
03b
Confirm & Pay booking-confirm.html
Session details grid, bay info, price breakdown, and an integrated payment form (card number, expiry/CVC). Trust badges and sidebar summary. No separate payment page.
Payment form Price breakdown Trust badges Sidebar summary
04
Dashboard dashboard.html
Authenticated user hub — greeting, 4-col stats grid, "Next Session" card with live indicator and countdown timer, upcoming sessions list, and session history table. Profile dropdown in topbar.
Stats grid Live indicator Countdown timer Session history Profile dropdown
05
Booking Confirmed confirmation.html
Post-booking celebration page — dark-green hero with checkmark and reference code, booking detail card with QR placeholder, "What's Next" guidance, and calendar/app download CTAs.
Dark hero banner Reference code QR placeholder What's Next card
06
Edit Profile profile.html
Minimal centered form card — avatar with membership tier badge, pre-filled name/email/phone fields, cancel and save actions.
Avatar + tier badge Profile form Centered card layout
